# ZeroTrace

**Keyed Unicode Watermarking Framework for AI Generated Text Provenance**

## Goal
Verify whether a text passed through a watermark-enabled AI generation pipeline. This is NOT an AI detector (does not use perplexity, stylometry, or linguistics).

## Features
- Post-generation, model-agnostic
- Cryptographically verifiable
- Provenance & authenticity verification
- Tamper detection
- Watermark recovery
- Integrity scoring
